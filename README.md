# Image Tools

## Overview

Simple image processing utilities using Python.

---

## Features

* Resize images
* Convert images to grayscale

---

## Files

* `resize.py` → resize images
* `grayscale.py` → convert to grayscale

---

## Usage

```bash
python resize.py
python grayscale.py
```

---

## Use Case

* Preprocessing for ML models
* Basic image manipulation
* CV pipelines
